# 🛡 ShieldCare Insurance Management UI

A modern and responsive Insurance Management Web Interface built using pure HTML, CSS, and JavaScript.

---

## 🚀 Project Overview

ShieldCare is a front-end web project that simulates a simple insurance management system.

The application allows users to:

- Login securely
- View insurance dashboard
- See policy statistics
- Apply for new insurance
- Receive modal confirmation



This project focuses on clean UI design, structured layout, and DOM interaction.

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox, Layout Structuring)
- JavaScript (DOM Manipulation & Events)
- Git & GitHub
